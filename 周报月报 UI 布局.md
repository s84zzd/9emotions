🌟 周报 UI 布局（Weekly = 近 7 天）
===========================

周报的核心是：**轻、即时、可快速浏览、关注波动**。  
布局节奏要短、快、清晰。

* * *

**📌 周报整体结构（移动端）**
------------------

    [顶部标题区]
      - “你的近 7 天”
      - 小副标题：轻柔一句话总结（自动生成）
    
    [图表区 1：情绪波动折线图]
      - 标题：近 7 天情绪波动
      - 折线图（高峰/低谷自动标注）
      - 图表说明文案（自动生成）
    
    [图表区 2：场景分布条形图]
      - 标题：近 7 天场景分布
      - 条形图（高亮 dominant scene）
      - 图表说明文案（自动生成）
    
    [模块 1：本周情绪概览]
      - Top emotions（标签样式）
      - 情绪趋势（上升/下降/混乱/平稳）
      - 正负能量比例（小圆环图或数字）
    
    [模块 2：本周场景洞察]
      - 场景卡片（work / study / social…）
      - 每个场景一句洞察
    
    [模块 3：本周亮点]
      - 2–3 条 bullet points
    
    [模块 4：本周挑战]
      - 1–2 条 bullet points
    
    [模块 5：下周建议（短期、小步）]
      - 2 条轻量建议
      - 建议卡片（可点击进入活动页）
    
    [底部 CTA]
      - “查看你的活动记录”
      - “继续记录情绪”

* * *

**📌 周报视觉节奏（给设计师）**
-------------------

* **图表在上，文案在下**（先看数据，再看解释）
* **图表占比约 40%**
* **文案模块占比约 60%**
* 色彩轻柔：粉雾紫、浅蓝、奶白
* 图表线条柔和，不要尖锐折角
* 场景卡片用轻色背景（如 #F7F4FF）

* * *

🌟 月报 UI 布局（Monthly = 当月）
=========================

月报的核心是：**深度、结构性、趋势、洞察**。  
布局节奏要稳、深、层次分明。

* * *

**📌 月报整体结构（移动端）**
------------------

    [顶部标题区]
      - “你的这个月”
      - 小副标题：一句深度总结（自动生成）
    
    [图表区 1：情绪结构玫瑰图]
      - 标题：本月情绪结构
      - 玫瑰图（9 情绪 × 半径 = 频率）
      - 图表说明文案（自动生成）
    
    [图表区 2：情绪趋势曲线]
      - 标题：本月情绪趋势
      - 趋势曲线（平滑）
      - 图表说明文案（自动生成）
    
    [模块 1：本月情绪结构]
      - 正负能量比例（大号数字）
      - 主导情绪（标签样式）
    
    [模块 2：本月趋势分析]
      - 情绪趋势（上升/下降/混乱/平稳）
      - 情绪模式（如“高压力 + 高恢复”）
    
    [模块 3：本月场景模式]
      - 场景卡片（主导场景高亮）
      - 每个场景一句深度洞察
    
    [模块 4：本月亮点]
      - 3 条 bullet points（偏深度）
    
    [模块 5：本月挑战]
      - 2 条 bullet points（结构性问题）
    
    [模块 6：下个月方向性建议（长期）]
      - 2 条方向性建议
      - 建议卡片（可跳转到活动页）
    
    [底部 CTA]
      - “查看你的完整记录”
      - “继续记录情绪”

* * *

**📌 月报视觉节奏（给设计师）**
-------------------

* **图表占比约 50%**（月报更数据驱动）
* **文案占比约 50%**
* 色彩更稳重：柔紫、雾蓝、浅灰
* 玫瑰图要柔和、圆润
* 趋势曲线要平滑，不要折线感太强
* 场景卡片可用更深的色块区分（如 #EDEAFF）

* * *

🌟 周报 vs 月报 UI 对比（可放进 PRD）
==========================

| 维度   | 周报（近 7 天） | 月报（当月）     |
| ---- | --------- | ---------- |
| 图表类型 | 折线图 + 条形图 | 玫瑰图 + 趋势曲线 |
| 图表目的 | 呈现波动      | 呈现结构与趋势    |
| 文案风格 | 轻、即时、贴近生活 | 深、洞察、总结性   |
| 建议类型 | 小步、可立即执行  | 方向性、长期改善   |
| 信息密度 | 低         | 中–高        |
| 视觉节奏 | 快、轻       | 稳、深        |
